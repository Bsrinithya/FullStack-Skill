
# Hibernate HQL Skill 3 Lab

This project demonstrates:

- HQL Sorting
- Pagination
- Aggregate Functions
- GROUP BY
- Price Range Filtering
- LIKE Pattern Matching

## Run Steps

1. Create database

CREATE DATABASE hibernate_lab;

2. Update credentials in hibernate.cfg.xml

3. Run ProductDataLoader once to insert sample data

4. Run HQLDemo to execute all queries
